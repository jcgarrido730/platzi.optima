Cambios en GitHub de master a main
Desde el 1 de octubre de 2020 GitHub cambió el nombre de la rama principal, ya no es "master" sino main.

Este derivado de una profunda reflexión ocasionada por el movimiento #BlackLivesMatter.

La industria de la tecnología lleva muchos años usando términos como master, slave, blacklist o whitelist y se espera que pronto puedan ir desapareciendo.
