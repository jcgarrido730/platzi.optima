# Mamba
https://mamba.readthedocs.io/en/latest/

## Instalar MANBA
```
$ conda install --channel conda-forge mamba
$ mamba --help
```

## Crear ambiente 
```
$ conda env list
$ mamba env create --file environment.yml
$ conda env list
```

# Activar ambiente
```
$ conda activate py39
```

## Desactivar ambiente
```
$ conda deactivate py39
```
