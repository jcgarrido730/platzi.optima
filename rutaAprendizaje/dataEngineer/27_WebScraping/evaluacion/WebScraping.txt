Curso de Web Scraping: Extracción de Datos en la Web

1. ¿Puedo tener alguna consecuencia si hago Web Scraping?
Sí, si se violan leyes informáticas o los Términos y Condiciones.

2. ¿Qué función debemos usar para solicitar el código HTML de una página?
requests.get(url)

3. ¿Cómo obtenemos un tag de un HTML parseado con BeautifulSoup?
Utilizando el método find y buscando por el nombre del tag y sus atributos.

4. ¿Cómo extraemos un atributo de un tag?
Utilizando el método get(atributo)

5. ¿Cómo podemos hacer para prevenir que una excepción interrumpa la ejecución del programa?
Usando la sentencia try, except.

6. ¿Si el contenido de una request no es texto, cómo es conveniente almacenarlo?
Debemos quedarnos con el atributo .content de la respuesta.

7. ¿Cuál es la ventaja de utilizar Scrapy?
Todas son correctas

8. ¿Por qué debemos cambiar de método para scrapear sitios dinámicos?
Porque necesitamos ejecutar código JS para acceder a la información.

9. ¿Cuál es el XPath correcto para seleccionar el tercer ítem de una lista que pertenece a la clase "flights"?
//li[@class="flight"][3]

10. ¿Cómo hacemos para abrir el modal (pop-up) con la información de las escalas?
Seleccionando el elemento del botón de escalas y luego clickeando.

11. ¿Cómo hacemos para seleccionar un elemento cuya clase contiene una palabra?
[contains(@class, "palabra")]

12. ¿Qué debemos hacer para asegurarnos que una página terminó de cargar lo que esperábamos?
Utilizar las funciones de Selenium para esperar la presencia de un elemento determinado.

13. ¿Qué es lo último que debo hacer cuando terminé de usar el driver?
.close()

14. ¿En qué casos debemos usar Selenium?
Cuando no nos quede alternativa.

15. ¿Qué es una API?
Una interfaz para programar aplicaciones.

16. ¿Por qué obtenemos un status code 401?
Porque no estamos enviando el token de autorización.

17. ¿Qué dato debemos incluir en el encabezado de cualquier solicitud para que nos autoricen la respuesta?
Depende de la solicitud. Hay que leer la documentación

18. Para acceder al endpoint de un álbum y conocer sus canciones:
Debemos conocer el id de ese álbum

19. ¿Qué significa usar un proxy?
Que hay un intermediario entre el cliente y el servidor.

20. ¿Para qué sirve Tesseract?
Para hacer reconocimiento óptico de caracteres a partir de una imagen.
