Curso de Fundamentos de Ingeniería de Datos

1. ¿Cuáles son las tres habilidades fundamentales de la ciencia de datos?
Programación, estadística y matemáticas y conocimiento temático (dominio del negocio)

2. ¿Cuáles son las tres habilidades más fuertes que debe tener una Data Engineer?
Programación, manipulación de datos e ingeniería de software.

3. ¿Cuál es el rol encargado generalmente de la parte exploratoria de un dataset para encontrar valor?
Data Scientist

4. ¿Cuál es el rol encargado generalmente de poner en producción pipelines de datos para analytics y modelos de machine learning?
Data Engineer

5. ¿Qué es DataOps?
Es una filosofia para el tratamiento de datos, desde su almacenamiento hasta su operación

6. ¿Cómo se le conoce al periodo de tiempo en Scrum para poder entregar una iteración?
Sprint

7. ¿Por qué Python es el principal lenguaje en ingeniería de datos actualmente?
Por su sencillez y amplia variedad de librerías.

8. ¿Cuál es una ventaja de codear en una Jupyter Notebook contra un editor de texto?
En la notebook puedes evaluar los resultados de una fracción de codigo dentro de la misma.

9. ¿Por qué es bueno esperar a tener tres observaciones/repeticiones de un proceso para automatizarlo?
Para entenderlo bien, saber que se repite y conocer cómo itera.

10. ¿Para qué sirve SQL en ingeniería de datos?
Es el lenguaje de facto para consultar datos en bases de datos con esa tecnología.

11. ¿Qué es Web Scraping y para qué sirve?
Es el proceso de obtener datos de una fuente de información, generalmente de una página web.

12. ¿Qué es Apache Spark y para qué se utiliza?
Es una tecnología que permite trabajar con un cluster para procesar datos en paralelo

14. ¿Cuál es una ventaja de usar Docker al trabajar con ambientes/computadores distintos?
Aislar el sistema operativo a un programa e instalar sus dependencias sin variar cómo se ejecuta.

15. Al trabajar con diferentes ambientes desde la planeación hasta la puesta en producción, ¿cuál es el ambiente donde se debe probar el software?
Desarrollo

16. ¿Cuáles son los tests de software que se encargan de verificar que una función esté funcionando como fue diseñada?
Los tests unitarios.

17. ¿Existe definitivamente un proveedor de servicios en la nube o cloud que es el mejor indiscutiblemente?
No, dependerá del equipo de trabajo y requerimientos del proyecto.

18. ¿Qué es el model drifting?
Cuando un modelo deja de funcionar en el tiempo por la variabilidad de las circunstancias

19. ¿Cuál es un medio importante de medir la salud de un servicio de datos en la nube?
Loggings y monitoreo

20. ¿La mentoría es una forma de aprender?
Verdadero, compartiendo aprendes más y refuerzas lo que conoces.

21. ¿Cuál es el rol que está 100% enfocado a mantener modelos en producción al que puede evolucionar un Data Engineer?
Machine Learning Engineer

22. ¿Cuál es el rol que está enfocado en diseñar y mantener una arquitectura de datos al que puede evolucionar una Data Engineer?
Data Architect

23. ¿Es necesario aprender todos los lenguajes de programación usados en ingeniería de datos?
Falso. Lo que importa es sentirse con comodidad implementando en diversos paradigmas de programación y llevar ideas a generar valor.
